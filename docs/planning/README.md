# Text-Based Project Management

Copendex keeps lightweight project management in text files so planning stays reviewable, searchable, and versioned with the code.

## Files

- [../roadmap.md](../roadmap.md): high-level direction and non-goals.
- [backlog.md](backlog.md): lightweight task inventory.
- [milestones.md](milestones.md): release-oriented planning.
- [../../CHANGELOG.md](../../CHANGELOG.md): user-visible changes by release.
- [../decisions](../decisions): architecture decision records.

## Workflow

1. Capture ideas in [backlog.md](backlog.md).
2. Promote scoped work into [milestones.md](milestones.md).
3. Record user-visible changes in [../../CHANGELOG.md](../../CHANGELOG.md).
4. Use decision records for durable technical choices.
5. Keep implementation review in pull requests.
